Leetcode 945: 使数组唯一的最小增量 [中等]

给定整数数组 A，每次 move 操作将会选择任意 A[i]，并将其递增 1。
返回使 A 中的每个值都是唯一的最少操作次数。

示例 1:
输入：[1,2,2]
输出：1
解释：经过一次 move 操作，数组将变为 [1, 2, 3]。
示例 2:
输入：[3,2,1,2,1,7]
输出：6
解释：经过 6 次 move 操作，数组将变为 [3, 4, 1, 2, 5, 7]。
可以看出 5 次或 5 次以下的 move 操作是不能让数组的每个值唯一的。

提示：
0 <= A.length <= 40000
0 <= A[i] < 40000

思路：用空间换时间

首先，统计每个数出现的次数，使用hash思想
然后将出现次数大于1的数move操作，
具体方法为将当前数a的值+1得到b，然后将b的出现次数+1，a出现的次数-1
将此过程一直循环到每个数出现的次数均小于等于1

注意：提示里说了数字的范围，最大需要80000
